# Agriplus PWA

A Gatsby site.

__FEATURES__

* [x] Custom logo
* [x] Custom favicon
* [ ] PWA offline support
* [ ] Gatsby images plugin
* [ ] SEO https://github.com/histaff/website-static/blob/master/src/components/seo.js
